Quick Start

Up and running in 30 seconds

# Clone
git clone https://github.com/kongshan001/one-person-company.git
cd one-person-company

# Run directly (Python 3.8+)
python3 products/paste-hut/server.py     # → http://localhost:9292
python3 products/ping-bot/monitor.py     # → http://localhost:8081

# Or with Docker
docker compose up -d                     # → All services

# Run tests
python3 -m unittest discover -s tests -v # → 50 tests
