# 💰 收入模型

## 单位经济

### IconForge（美术资产工厂）

```
单张图标成本:
├── 文生图 API 调用:     $0.001-0.01/张 (Pollinations 免费)
├── Prompt 生成:         $0.0005/张
├── 质检 + 后处理:       $0.0002/张
└── 总成本:              ≈ $0.002/张

定价:
├── 订阅 $9.9/月 → 100 张/月 = $0.099/张 → 毛利 98%
├── 订阅 $19.9/月 → 300 张/月 = $0.066/张 → 毛利 97%
└── 一次性 $29.9 → 500 张 = $0.060/张 → 毛利 97%
```

### SaaS 工具

```
PasteHut:
├── 免费版: 10 paste/天, 7天过期
├── Pro $5/月: 无限 paste, 自定义过期, 密码保护
├── 服务器成本: ~$3/月 (共享)
└── 100 用户时: $500 收入 / $3 成本 = 99% 毛利

PingBot:
├── 免费版: 3 个监控点, 5分钟间隔
├── Pro $9.9/月: 50 个监控点, 1分钟间隔, 飞书/邮件告警
├── 服务器成本: ~$5/月
└── 50 用户时: $495 收入 / $5 成本 = 99% 毛利
```

```

盈亏平衡: 50 / 9.9 ≈ 6 个付费用户

→ 极低门槛，第 2 个月即可盈亏平衡
```
