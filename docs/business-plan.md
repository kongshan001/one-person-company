# 📋 商业计划书

## 愿景

用 AI Agent 替代传统公司的执行层，一个人运营一家有完整产品线的公司。

## 市场分析

### 美术资产市场
- 全球游戏美术外包市场：$50B+ (2024)
- 独立游戏开发者最缺：时间和美术资源
- 现有方案：手动绘制($50-200/图标)、素材包(贵且不定制)
- **我们的优势**：AI 生成 → 秒级出图 → 价格 1/10

### 开发者工具市场
- 全球 SaaS 市场：$195B+ (2024)
- 开发者工具细分：$20B+
- **我们的优势**：零依赖、极简部署、AI 原生

## 收入目标

| 阶段 | 时间 | 月收入目标 | 里程碑 |
|------|------|-----------|--------|
| 🌱 种子期 | 月 1-3 | $0 → $100 | 产品上线，首个付费用户 |
| 🌿 成长期 | 月 4-6 | $100 → $500 | 3 个产品，10+ 付费用户 |
| 🌸 扩张期 | 月 7-12 | $500 → $2000 | 自动化运营，被动收入为主 |
| 🍯 成熟期 | 月 12+ | $2000 → $5000+ | 产品矩阵，订阅为主 |

## 收入来源

```
总收入 = 产品订阅 + 资产包销售 + 定制服务 + Agent 托管

├── 产品订阅 (40%)
│   ├── IconForge:  $9.9/月 × N 用户
│   ├── PasteHut:   $5/月 × N 用户
│   └── PingBot:    $9.9/月 × N 用户
│
├── 资产包销售 (30%)
│   ├── 单包:       $5-30/包
│   └── 定制包:     $50-200/单
│
├── 定制服务 (20%)
│   └── 特殊风格/大批量: $100-500/单
│
└── Agent 托管 (10%)
    └── 为客户部署 AI Agent: $99-499/月
```

## 成本结构

| 项目 | 月成本 | 说明 |
|------|--------|------|
| VPS | $5-20 | 1-2 台轻量服务器 |
| 域名 | $1 | 年均 $10-15 |
| 文生图 API | $0-50 | 初期用免费额度 |
| LLM API | $10-30 | Hermes + 生成 Prompt |
| 支付处理 | 收入的 3-5% | Stripe / LemonSqueezy |
| **合计** | **$16-101/月** | 极低运营成本 |

## 竞争策略

1. **价格** — AI 生成，成本接近零，定价是传统 1/10
2. **速度** — 秒级出图 vs 传统 1-7 天
3. **定制** — 每次生成都是独一无二的
4. **迭代** — 不满意？1 秒重新生成
5. **AI 原生** — 从第一天就用 AI 运营，不是后加的

## 风险与应对

| 风险 | 概率 | 影响 | 应对 |
|------|------|------|------|
| 文生图质量不稳定 | 高 | 中 | AI 质检 + 人工抽检 |
| API 成本上涨 | 中 | 中 | 多供应商备选 + 本地模型 |
| 被大厂复制 | 低 | 高 | 专注细分 + 速度优势 |
| 客户信任度 | 中 | 中 | 样品免费 + 口碑传播 |
